This project involves the creation of a Discord bot using Node.js and Express. The bot can respond to slash commands and interact with people in a Discord server. Initially, the bot was built with only one function that was to perform a simple ping/pong action. Over time, the bot has been upgraded with customized functions and commands.
Features
/ping → Responds “pong”
/complete → Autocomplete words from a list
/joke → Send a coding joke randomly
/about → Provide details about the bot

How It Works

The Discord bot uses an Express server to listen for interactions on Discord. Once the user inputs a slash command, the bot analyzes which command was used and then responds accordingly.
